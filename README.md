    cd apktool_v2.4.1
    chmod +x *
    sudo ./install.sh or sudo bash install.sh

Install Missing Packages manually or run

type:

    chmod +x install.sh
    sudo ./install.sh or sudo bash install.sh

It will install all packages that are missing and it will automaticaly start...

    sudo ./evil
